# Customer Churn Prediction Dashboard

A full-stack web application that predicts customer churn risk using a trained machine learning model, visualizes churn analytics, and logs prediction history — built as part of an internship project.

## Overview

This dashboard lets a user:
- View live analytics on a customer base (churn rate, satisfaction, subscription breakdown, correlations)
- Enter a customer's profile and get an instant churn risk prediction with a confidence gauge
- Receive contextual retention recommendations based on the prediction
- View a history of past predictions
- Sign up / log in to access the prediction feature (auth-gated action)
